Request a document

Use this service to ask us to add a document to a company's filing history if it is not currently available.

How long it takes

It takes us up to 2 hours to check the availability of a document and add it to a company's filing history between 8.30am - 3pm, Monday - Friday (excluding holidays).

If you make a request outside of this time, the document will be added the next working day.

After you have made the request, you can check the company's filing history for updates on the document's availability.

Requesting a document takes around 5 minutes.
